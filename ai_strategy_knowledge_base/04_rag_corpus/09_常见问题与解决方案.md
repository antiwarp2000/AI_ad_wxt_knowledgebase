# 09_常见问题与解决方案

> **文件定位**：投放类、账户类、数据类常见问题速查表。AI 在回答"计划跑不动""账户被限制""数据延迟"等问题时查阅。

---

## 9.1 投放类问题

| 问题 | 可能原因 | 解决方案 | 优先级 |
|------|---------|---------|-------|
| **计划跑不动** | 出价低/定向窄/预算低 | 提高出价10%-20%/扩大定向/增加预算 | 高 |
| **计划花费太快** | 出价高/定向宽/匹配宽 | 降低出价10%-20%/缩小定向/精确匹配 | 高 |
| **ROI不达标** | CPC高/转化低/客单低 | 优化质量分/优化详情页/提高客单 | 高 |
| **CTR低** | 创意差/关键词不相关 | 优化创意/调整关键词 | 中 |
| **转化率低** | 流量不精准/商品问题 | 优化定向/优化商品 | 中 |
| **无展现** | 出价低/预算低/定向窄/商品下架 | 排查6.1无展现异常 | 高 |
| **有展现无点击** | 创意差/价格高/评价差 | 排查6.2有展现无点击异常 | 中 |
| **有点击无转化** | 详情页差/流量不精准 | 排查6.3有点击无转化异常 | 中 |
| **CPC持续上涨** | 竞争加剧/质量分下降 | 优化质量分/降低出价 | 中 |
| **预算频繁撞线** | 预算设置过低 | 增加预算30%-50% | 低 |

## 9.2 账户类问题

| 问题 | 可能原因 | 解决方案 | 优先级 |
|------|---------|---------|-------|
| **账户被限制** | 违规操作/素材违规/投诉 | 检查违规记录、修改后申诉 | 高 |
| **余额不足** | 未及时充值 | 设置余额预警、及时充值 | 高 |
| **多计划互相竞争** | 关键词/人群重叠 | 错开关键词/人群、分计划投放 | 中 |
| **计划无法创建** | 资质不全/商品违规 | 补充资质、处理违规 | 高 |
| **创意审核不通过** | 素材违规/文字超标 | 修改后重新提交 | 中 |
| **权限不足** | 子账号权限限制 | 主账号授权 | 低 |

---

## 9.3 数据类问题

| 问题 | 可能原因 | 解决方案 | 优先级 |
|------|---------|---------|-------|
| **数据延迟** | 系统统计延迟 | 等待24小时后查看 | 低 |
| **数据异常波动** | 系统故障/大促影响 | 联系阿里妈妈客服确认 | 中 |
| **转化数据不准** | 归因周期/跨店问题 | 了解归因规则、合理评估 | 低 |
| **报表数据不一致** | 统计口径不同 | 确认报表口径(实时vs离线) | 低 |
| **搜索词报告缺失** | 数据量不足 | 积累数据后查看 | 低 |
| **人群报告无数据** | 人群覆盖过小 | 扩大人群范围 | 低 |

## 9.4 紧急处理流程

### P0级问题(立即处理)
- 账户被限制 → 检查违规、申诉
- 余额不足 → 立即充值
- 商品下架 → 恢复上架

### P1级问题(2小时内处理)
- 计划跑不动 → 调整出价/定向
- 花费过快 → 降低出价/缩小定向
- ROI严重不达标 → 暂停计划、诊断

### P2级问题(24小时内处理)
- CTR低 → 优化创意
- 转化率低 → 优化详情页/流量
- 数据异常 → 联系客服确认

### P3级问题(本周内处理)
- 预算频繁撞线 → 调整预算
- 数据延迟 → 等待系统更新
- 报表不一致 → 确认口径
