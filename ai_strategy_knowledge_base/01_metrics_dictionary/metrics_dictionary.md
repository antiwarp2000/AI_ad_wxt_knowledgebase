# 指标字典与字段口径

## 核心指标

| 指标 | 公式 | 用途 | 备注 |
|---|---|---|---|
| 花费 | 原始报表消耗字段 | 成本控制 | 单位：元 |
| 成交金额 / GMV | 原始报表成交金额字段 | 收入规模 | 单位：元 |
| ROI | 成交金额 ÷ 花费 | 效果判断核心指标 | 花费为 0 时不可直接计算 |
| 展现量 | 原始报表展现字段 | 流量获取 | 用于判断无展现/低展现 |
| 点击量 | 原始报表点击字段 | 流量质量 | 用于 CTR/CPC |
| CTR | 点击量 ÷ 展现量 | 创意/相关性判断 | 展现为 0 时不可计算 |
| CPC | 花费 ÷ 点击量 | 点击成本 | 点击为 0 时不可计算 |
| 支付订单数 | 原始报表订单字段 | 转化结果 | 用于无成交判断 |
| CVR | 支付订单数 ÷ 点击量 | 转化承接能力 | 点击为 0 时不可计算 |
| CPA / 成交成本 | 花费 ÷ 支付订单数 | 成本效率 | 订单为 0 时不可计算 |
| 预算消耗率 | 实际花费 ÷ 日预算 | 预算利用效率 | 需要计划日预算字段 |

## 默认参考阈值

| 指标 | 健康 | 亚健康 | 不健康 |
|---|---|---|---|
| ROI | ≥ 目标 ROI | 目标 ROI 的 80%-100% | < 目标 ROI 的 80% |
| CTR | ≥ 行业均值 | 行业均值的 70%-100% | < 行业均值的 70% |
| CVR | 稳定或上升 | 波动较大 | 持续下降或明显低于均值 |
| CPC | 稳定或下降 | 持续上涨 | 高于行业/账户均值 150% |
| 预算消耗率 | 70%-95% | 50%-70% 或接近撞线 | <50% 或 >95% |

## 字段映射建议

后续接入新报表时，先把平台字段映射到标准字段：

```json
{
  "date": "日期",
  "plan_id": "计划ID",
  "plan_name": "计划名称",
  "spend": "花费",
  "gmv": "成交金额",
  "impressions": "展现量",
  "clicks": "点击量",
  "orders": "支付订单数",
  "budget": "日预算"
}
```
