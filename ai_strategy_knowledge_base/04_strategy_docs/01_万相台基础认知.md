## 1.1 产品定位与架构

### 产品定位
- **万相台无界版**：阿里妈妈一站式智能投放平台，整合原直通车、引力魔方、万相台旧版能力
- **核心价值**：通过 AI 智能算法，实现"人-货-场"精准匹配，降低投放门槛，提升 ROI

### 核心能力矩阵
| 能力模块 | 对应场景 | 核心功能 |
|---------|---------|---------|
| 关键词推广 | 搜索流量获取 | 关键词选词、出价、匹配方式管理 |
| 精准人群推广 | 推荐流量获取 | 达摩盘人群定向、Lookalike扩展 |
| 货品加速 | 商品全生命周期 | 新品加速、爆款打造、清仓推广 |
| 消费者运营 | 用户资产运营 | 拉新、促活、会员转化 |
| 内容营销 | 内容流量获取 | 短视频、直播、图文引流 |

### 投放场景分类
| 场景 | 适用阶段 | 核心目标 |
|------|---------|---------|
| 拉新快 | 全阶段 | 获取新客、扩大人群资产 |
| 上新快 | 新品期 | 新品冷启动、快速测款 |
| 货品加速 | 成长期/成熟期 | 商品成交规模最大化 |
| 活动加速 | 大促期 | 活动期间爆发成交 |
| 会员快 | 成熟期 | 会员招募与复购 |
| 内容快 | 全阶段 | 内容种草与引流 |

## 1.2 投放底层逻辑

### 竞价机制
```
eCPM = 出价 × 预估CTR × 预估CVR × 1000
```
- **eCPM**：千次展现收益，决定广告排名
- **出价**：商家愿意为一次点击支付的最高金额
- **预估CTR**：系统预测的点击率（受创意质量、相关性影响）
- **预估CVR**：系统预测的转化率（受商品竞争力、详情页影响）

### 流量分发逻辑
| 流量类型 | 获取方式 | 特点 |
|---------|---------|------|
| 搜索流量 | 关键词竞价 | 意图明确、转化率高、竞争激烈 |
| 推荐流量 | 人群定向 | 流量大、成本低、转化率波动大 |
| 内容流量 | 内容营销 | 种草属性强、转化周期长 |

### 智能出价算法
| 出价方式 | 原理 | 适用场景 |
|---------|------|---------|
| 目标ROI出价 | 系统自动调整出价以达到目标ROI | 有历史转化数据、追求稳定ROI |
| 最大化拿量 | 在预算内尽可能获取更多转化 | 预算充足、追求规模 |
| 控成本投放 | 严格控制单次转化成本不超过设定值 | 成本敏感、预算有限 |

### 人群定向逻辑
- **达摩盘人群包**：基于阿里数据资产的人群标签组合
- **系统智能人群**：算法自动匹配高转化概率人群
- **自定义人群**：商家自行圈选的人群组合
- **人群溢价**：在基础出价上对特定人群加价获取

## 1.3 账户结构与层级

### 四级结构
```
账户 (Account)
└── 计划 (Campaign)
    └── 单元 (AdGroup)
        └── 创意 (Creative)
```

### 各层级配置项
| 层级 | 可配置项 | 权限边界 |
|------|---------|---------|
| 账户 | 账户预算、充值、权限管理 | 全局生效 |
| 计划 | 投放目标、日预算、投放时间/地域、出价方式 | 计划内生效 |
| 单元 | 商品/关键词/人群、出价、匹配方式 | 单元内生效 |
| 创意 | 主图、标题、视频、落地页 | 创意级生效 |

### 预算分配层级
| 层级 | 说明 | 优先级 |
|------|------|-------|
| 账户预算 | 账户总消耗上限 | 最高（全局限制） |
| 计划预算 | 单个计划日消耗上限 | 中（计划内限制） |
| 单元预算 | 单个单元消耗分配 | 低（系统自动分配） |

### 计划类型与场景对应
| 计划类型 | 适用场景 | 推荐出价方式 |
|---------|---------|-------------|
| 关键词推广 | 搜索流量获取 | 目标ROI出价 / 手动出价 |
| 精准人群推广 | 推荐流量获取 | 目标ROI出价 / 最大化拿量 |
| 货品加速 | 商品推广 | 目标ROI出价 |
| 消费者运营 | 拉新/促活 | 控成本投放 |

## 1.4 关键概念速查

| 概念 | 定义 | 备注 |
|------|------|------|
| eCPM | 千次展现收益，决定广告排名 | eCPM = 出价 × 预估CTR × 预估CVR × 1000 |
| 质量分 | 关键词质量评分(1-10分) | 影响实际扣费和排名 |
| 冷启动期 | 计划新建后0-7天的学习期 | 数据波动大，避免频繁修改 |
| 预算撞线 | 日预算消耗完毕，计划暂停投放 | 可能损失后续流量 |
| 人群溢价 | 对特定人群在基础出价上加价 | 溢价比例通常10%-300% |
| 匹配方式 | 关键词与搜索词的匹配规则 | 精确/短语/广泛 |
| 归因周期 | 转化数据归属的时间窗口 | 通常15天或30天 |
