# 03_投中执行阶段

> **文件定位**：计划创建、冷启动期管理、稳定期管理、大促期管理。AI 在回答"计划怎么建""冷启动没数据怎么办""大促怎么调整"等问题时查阅。

## 3.1 计划创建与配置

### 计划创建流程
```
1. 选择投放场景 → 2. 设定投放目标 → 3. 配置预算与出价 → 4. 圈选商品/人群/关键词 → 5. 绑定创意 → 6. 提交审核
```

### 关键配置项检查清单
| 配置项 | 检查内容 | 常见错误 |
|--------|---------|---------|
| 投放时间 | 全天投放 vs 自定义时段 | 忘记设置导致非高峰浪费 |
| 投放地域 | 全地域 vs 自定义地域 | 未排除不发货地区 |
| 终端设置 | 无线/PC/全部 | 未优先无线(流量占比90%+) |
| 加速投放 | 加速 vs 标准 | 冷启动误用加速导致花费过快 |
| 预算设置 | 日预算金额 | 预算过低导致撞线 |
| 出价方式 | 目标ROI/最大化拿量/控成本 | 无数据时使用目标ROI |

## 3.2 冷启动期管理（0-7天）

### 冷启动特征
| 特征 | 说明 | 是否正常 |
|------|------|---------|
| 数据波动大 | 展现/点击/转化每日变化大 | ✅ 正常 |
| 展现量爬坡 | 从少到多逐步增加 | ✅ 正常 |
| CPC偏高 | 初期点击成本高于稳定期 | ✅ 正常 |
| 无数据 | 24小时内完全无展现 | ⚠️ 需排查 |

### 冷启动期操作原则
| 原则 | 说明 | 违反后果 |
|------|------|---------|
| 避免频繁修改 | 每次修改可能重置学习期 | 延长冷启动时间 |
| 预算充足 | 日预算 ≥ 50个点击成本 | 数据不足，模型无法学习 |
| 观察周期 | 至少3-5天再评估效果 | 过早判断导致误删计划 |
| 不频繁调价 | 调价幅度≤20%，间隔≥2天 | 模型无法收敛 |

### 冷启动异常处理决策树
```
冷启动异常排查
├─ 无展现
│  ├─ 检查出价 → 低于市场均价 → 提高出价10%-20%
│  ├─ 检查预算 → 预算过低 → 增加预算
│  ├─ 检查定向 → 范围过窄 → 扩大人群/关键词
│  └─ 检查商品 → 下架/违规 → 恢复上架
│
├─ 有展现无点击
│  ├─ 检查创意 → CTR低于行业 → 优化主图/标题
│  ├─ 检查相关性 → 关键词与商品不匹配 → 暂停不相关词
│  └─ 检查价格 → 高于同行 → 调整价格或设置优惠
│
└─ 有点击无转化
   ├─ 检查详情页 → 转化要素缺失 → 优化详情页
   ├─ 检查评价 → 差评置顶 → 处理差评
   ├─ 检查流量 → 关键词/人群不精准 → 优化定向
   └─ 检查竞品 → 竞品有更强促销 → 调整策略
```

## 3.3 稳定期管理（7-30天）

### 数据监控频率
| 指标类型 | 监控频率 | 核心指标 |
|---------|---------|---------|
| 核心指标 | 每日 | 花费、展现、点击、CTR、转化、ROI |
| 趋势指标 | 每周 | 7日趋势、环比变化、渠道对比 |
| 深度指标 | 每月 | 人群画像、关键词效果、创意生命周期 |

### 优化动作清单
| 优化维度 | 操作 | 触发条件 |
|---------|------|---------|
| 关键词优化 | 加词/删词/调价 | 搜索词报告发现高转化词/低效词 |
| 人群优化 | 溢价调整/人群包替换 | 人群ROI差异>30% |
| 创意优化 | 替换低CTR创意/测试新素材 | CTR低于行业均值 |
| 出价优化 | 根据ROI调整出价 | ROI连续3天不达标/超标 |

### 预算调整策略
| 场景 | 判断条件 | 操作 |
|------|---------|------|
| 加预算 | ROI达标 + 预算撞线(消耗率>95%) | 每次增加20%-50% |
| 减预算 | ROI不达标 + 预算充足 | 每次减少10%-20% |
| 保持 | ROI达标 + 预算未撞线 | 维持现状，持续观察 |
| 扩定向 | 预算充足但花不出去 | 扩大人群/关键词范围 |
| 提出价 | 预算充足但花不出去 | 提高出价10%-20% |

## 3.4 大促期管理

### 大促节奏规划
| 阶段 | 时间 | 核心目标 | 预算策略 | 出价策略 |
|------|------|---------|---------|---------|
| 蓄水期 | 大促前7-15天 | 加购收藏 | 日常1.5倍 | 日常价 |
| 预热期 | 大促前1-7天 | 收藏转化 | 日常2倍 | +10% |
| 爆发期 | 大促当天 | 成交ROI | 日常3-5倍 | +20%-30% |
| 返场期 | 大促后1-3天 | 尾货清理 | 日常1.5倍 | 日常价 |

### 大促配置调整清单
| 配置项 | 调整内容 | 注意事项 |
|--------|---------|---------|
| 预算 | 提升至日常2-5倍 | 提前充值，避免余额不足 |
| 出价 | 提升10%-30% | 根据竞争态势动态调整 |
| 人群 | 放宽定向范围 | 增加行业高潜人群 |
| 创意 | 更新大促氛围素材 | 提前审核，避免活动当天未过审 |
| 关键词 | 增加活动词/促销词 | 如"双11""618""满减" |

### 大促期间监控要点
| 监控项 | 频率 | 异常处理 |
|--------|------|---------|
| 预算消耗 | 每2小时 | 消耗过快→限价；消耗过慢→提价 |
| ROI波动 | 每4小时 | ROI下滑→检查转化/客单 |
| 竞品动态 | 每日 | 竞品降价→调整策略 |
| 库存状态 | 每日 | 库存不足→暂停/降预算 |
