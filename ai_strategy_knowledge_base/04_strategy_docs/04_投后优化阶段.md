# 04_投后优化阶段

> **文件定位**：数据复盘框架、关键词/人群/创意/出价优化方法。AI 在回答"投放后怎么复盘""关键词怎么优化""人群怎么调整"等问题时查阅。

---

## 4.1 数据复盘框架

### 复盘维度矩阵
| 维度 | 分析内容 | 输出结论 |
|------|---------|---------|
| 时间维度 | 日/周/月/活动周期趋势 | 最佳投放时段、周期规律 |
| 商品维度 | 单品/品类/全店对比 | 高ROI商品、低效商品 |
| 渠道维度 | 搜索/推荐/内容对比 | 最优渠道、渠道预算分配 |
| 人群维度 | 新客/老客/会员对比 | 高价值人群、低效人群 |

### 复盘指标体系
| 指标类别 | 核心指标 | 分析目的 |
|---------|---------|---------|
| 消耗类 | 花费、CPC、CPM | 成本控制评估 |
| 流量类 | 展现、点击、CTR | 流量获取能力评估 |
| 转化类 | 加购、收藏、支付转化 | 转化承接能力评估 |
| 效果类 | ROI、CPA、GMV | 投入产出效率评估 |

### 复盘报告结构
```
1. 核心结论摘要（100-300字）
2. 数据对比分析
   - 环比对比（vs 上一周期）
   - 同比对比（vs 去年同期）
   - 目标对比（vs 设定KPI）
3. 问题诊断与归因
   - 异常指标识别
   - 根因分析
4. 优化建议与行动计划
   - 立即可做（本周内）
   - 需要进一步评估（下周内）
```

## 4.2 关键词优化

### 关键词诊断矩阵
| 展现 | 点击 | 转化 | 诊断 | 操作 |
|------|------|------|------|------|
| 高 | 高 | 高 | 核心优质词 | 保持或加价5%-10% |
| 高 | 高 | 低 | 流量词转化差 | 优化详情页或降价10%-20% |
| 高 | 低 | - | 创意/相关性差 | 优化创意或降价/暂停 |
| 低 | - | - | 出价低/匹配窄 | 提高出价10%-20%或改广泛 |
| 无 | - | - | 无流量 | 删除或大幅提价测试 |

### 关键词操作规范
| 操作 | 触发条件 | 操作幅度 | 注意事项 |
|------|---------|---------|---------|
| 加词 | 搜索词报告发现高转化词 | 每次≤20个 | 先小预算测试 |
| 删词 | 7天无展现 或 30天ROI严重不达标 | - | 删除前确认无潜力 |
| 调价 | ROI连续3天偏离目标 | ±10%-20% | 避免剧烈波动 |
| 改匹配 | 转化数据积累后调整 | 广泛→短语→精确 | 根据转化精准度 |

### 搜索词报告利用
| 分析项 | 操作 |
|--------|------|
| 高转化搜索词 | 添加为关键词 |
| 高花费无转化搜索词 | 添加为否定词 |
| 低相关搜索词 | 添加为否定词 |
| 竞品品牌搜索词 | 评估是否投放竞品词 |

## 4.3 人群优化

### 人群效果评估
| 评估维度 | 指标 | 判断标准 |
|---------|------|---------|
| ROI表现 | 各人群包ROI | ROI≥目标：优质；ROI<目标0.8：低效 |
| 转化成本 | CPA对比 | CPA≤目标：健康；CPA>目标1.5：过高 |
| 覆盖度 | 人群覆盖用户数 | 覆盖<1万：过窄；覆盖>100万：过宽 |
| 频次 | 人均触达次数 | 频次>5：可能疲劳 |

### 人群优化动作
| 场景 | 操作 | 幅度 |
|------|------|------|
| 高ROI人群 | 提高溢价 | +10%-30% |
| 高ROI人群 | 扩展Lookalike | 相似度1%-3% |
| 低ROI人群 | 降低溢价 | -10%-30% |
| 低ROI人群 | 暂停投放 | ROI<目标0.5且持续7天 |
| 新人群测试 | 小预算测试 | 日预算10%-20% |
| 人群疲劳 | 更换人群包 | 频次>5时 |

## 4.4 创意优化

### 创意效果评估标准
| 指标 | 优秀 | 良好 | 待优化 |
|------|------|------|-------|
| CTR | >行业均值1.5倍 | 行业均值1-1.5倍 | <行业均值 |
| CVR | >商品平均转化率 | ≈商品平均转化率 | <商品平均转化率 |
| CPC | <账户平均CPC | ≈账户平均CPC | >账户平均CPC |

### 创意优化动作
| 问题 | 操作 | 说明 |
|------|------|------|
| 低CTR | 更换主图、优化标题 | 突出卖点、差异化 |
| 高CTR低CVR | 检查商品承接页 | 流量精准但转化差 |
| 创意老化 | 定期更新素材 | 每2-4周更新一次 |
| 素材积累 | 建立高点击素材库 | 复用成功元素 |

### 创意生命周期管理
| 阶段 | 特征 | 操作 |
|------|------|------|
| 测试期(0-7天) | CTR波动大 | 保持观察，不频繁替换 |
| 稳定期(7-30天) | CTR稳定 | 持续投放，监测衰减 |
| 衰退期(30天+) | CTR持续下降 | 准备新素材替换 |

## 4.5 出价与预算优化

### 出价优化策略
| 出价方式 | 优化方法 | 适用条件 |
|---------|---------|---------|
| 目标ROI出价 | 根据实际ROI调整目标值 | 有≥15个转化数据 |
| 最大化拿量 | 预算充足时使用 | 追求规模、不严格控成本 |
| 控成本投放 | 设置CPA上限 | 成本敏感场景 |

### 出价调整规则
| 场景 | 操作 | 幅度 |
|------|------|------|
| ROI连续3天>目标1.2倍 | 提高出价扩量 | +5%-10% |
| ROI连续3天<目标0.8倍 | 降低出价控成本 | -10%-20% |
| CPC持续上涨 | 优化质量分或降价 | -5%-10% |
| 预算撞线且ROI达标 | 提高出价+加预算 | +10%出价，+30%预算 |

### 预算优化策略
| 策略 | 说明 | 适用场景 |
|------|------|---------|
| 预算倾斜 | 向高ROI计划倾斜预算 | 多计划并行时 |
| 分时预算 | 高峰时段高预算、低谷低预算 | 转化时段明显时 |
| 预算预警 | 设置消耗预警阈值(80%) | 防止超支 |

*本文件为投后优化模块，优化动作需遵循"小步快跑"原则，每次调整幅度控制在10%-20%。*
