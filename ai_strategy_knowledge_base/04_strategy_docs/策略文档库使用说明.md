# 策略文档库使用说明

这里不做向量索引，也不要求复杂的 RAG 流程。这个目录的作用是：把投放 SOP、策略经验、平台规则和排查说明按主题存放，方便人和 AI 按文件名直接读取。

## 适合放入这里的内容

- 投放前准备 SOP
- 投中调整方法
- 投后复盘方法
- 不同场景投放策略
- 常见问题排查方法
- 平台工具使用说明
- 合规和风险控制说明
- 你自己总结的运营经验

## 不适合只放在这里的内容

| 内容 | 应该放哪里 |
|---|---|
| ROI、CTR、CVR 公式 | `01_metrics_dictionary` |
| 黄金/潜力/止损/问题计划判断阈值 | `02_rules_engine` |
| 无展现/无点击/无转化排查路径 | `03_decision_trees` |
| 真实调整记录和结果 | `05_case_library` |
| 今天具体要执行的动作 | `06_action_playbooks` |

## 推荐文档写法

一篇文档尽量只讲一个主题，标题直接说明用途。

推荐格式：

```markdown
# 有点击无转化排查 SOP

## 适用场景

点击量正常，但订单数低或为 0。

## 判断依据

| 指标 | 判断方式 |
|---|---|
| 点击量 | 点击量 > 0 |
| 订单数 | 订单数 = 0 或 CVR 明显低于账户均值 |

## 常见原因

| 原因 | 判断方式 | 动作 |
|---|---|---|
| 流量不精准 | CTR正常但CVR低 | 收窄人群/关键词 |
| 价格弱 | 同款竞品更便宜 | 增加优惠或调整价格 |

## 今日动作

1. 暂停高花费无成交人群。
2. 检查详情页首屏和评价。
3. 观察次日 CVR 和 ROI 是否改善。
```

## 命名建议

- `有点击无转化排查SOP.md`
- `ROI下滑排查SOP.md`
- `新品投放策略.md`
- `活动期预算调整策略.md`
- `低ROI计划止损策略.md`

## 维护原则

- 标题要清楚，方便按文件名找到。
- 重要判断不要只写在长段落里，要用表格或列表写清楚。
- 如果一篇文档太长，就拆成多篇小文档。
- 策略文档只负责解释和方法，最终判断仍以规则库为准。
