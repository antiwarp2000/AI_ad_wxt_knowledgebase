# 示例案例库

## case_sample_001：有花费无成交的止损计划

- 计划分层：止损计划
- 异常类型：有点击无转化
- 触发条件：`spend > 0 and orders == 0`
- 诊断路径：点击存在，但无成交，优先排查流量精准度和商品承接。
- 动作建议：暂停高花费低效人群，检查详情页、评价、价格和优惠，必要时降低预算观察。
- 复盘指标：次日 CVR、ROI、成交成本是否改善。

## case_sample_002：ROI 达标的黄金计划

- 计划分层：黄金计划
- 异常类型：无明显异常
- 触发条件：`roi >= target_roi * 1.2 and orders > 0`
- 诊断路径：先确认预算是否撞线，再确认转化是否稳定。
- 动作建议：预算增加 10%-30%，优先放量高转化人群/关键词。
- 复盘指标：放量后 ROI 是否保持在目标以上。
