# 动作剧本库

## 黄金计划

目标：稳住 ROI 的前提下扩大成交。

动作：

1. 若预算撞线，预算增加 10%-30%。
2. 复制高 ROI 计划结构，用小预算测试相似商品/人群。
3. 保留高转化关键词和人群，避免频繁大改。
4. 观察放量后 ROI 是否回落，若明显回落则回退预算。

## 潜力计划

目标：补短板，把接近达标的计划推到达标。

动作：

1. CTR 低：优化主图、标题、利益点表达。
2. CVR 低：优化详情页、评价、SKU、优惠承接。
3. CPC 高：优化出价、提高质量分、收窄低效流量。
4. 连续 2-3 天改善后再小幅加预算。

## 止损计划

目标：防止继续消耗预算。

动作：

1. 立即降低预算或暂停。
2. 排查是否有点击无成交、高花费无转化、人群过泛。
3. 对高花费无转化关键词/人群做暂停或否定。
4. 未找到根因前不要加预算。

## 问题计划

目标：定位异常链路，先修复再扩量。

动作：

1. 无展现：查状态、预算、出价、定向、商品。
2. 有展现无点击：查创意、标题、价格、相关性。
3. 有点击无转化：查流量精准度、详情页、评价、竞品。
4. ROI 下滑：拆解 CPC、CVR、客单价三个因素。
